# Gym tracker

I got unreasonably angry at 50.000 variations on the same gym app charging a monthly subscription for a bloated ux nightmare. So I made it myself. Built with React, Typescript and Supabase. 

Access to my instance is whitelisted.

## Features

- **Workout Tracking**: Log exercises, sets, reps, and weights in real-time
- **Progress Analytics**: Track your progress over time with detailed insights and weight progression
- **Workout History**: View past workouts and exercise data, opt-in sharing to compare to other users.
- **User Authentication**: Sign-in with Supabase
- **Offline Support**: Works as a PWA for offline access
